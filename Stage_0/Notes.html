When the user sees a page through the browser, this one makes requests (HTTP) to a server via the Internet. The server responds with a file which is displayed in the browser.

  • Browsers: Firefox, Chrome, Safari, Internet Explorer
  • Servers: powerful computers hosting files
  • HTTP Protocol.

HTML

HTML HyperText Markup Language (brings everything together: text, images, videos, music, html) is the main type of document on the Internet.

Hyper links or links, link to other pages.

The markup makes the text look different and it's made of tags.

<NAME> content </NAME> -> this is called an element

Tags

Bold b
Italic em

It is said, in the world of programming, that computers are stupid because of the way they follow the human instructions.

For instance, if a closing tag is missing, the computer will render everything after the opening tag the same way.

Attributes

An attribute belongs to the opening tag and has a name and a value.

Anchor a, has an attribute called href and its value can be a URL. When rendered only the content of the tag would be displayed and this is called a link
Image img, has an attribute called src with URL as value. Another attribute is alt, which displays and alternative text if the image is not found. This tag is special, it's a VOID tag, as it has not content.

"This text    in HTML" renders to "This text in HTML" in the browser. All white spaces turn into a single white space.

br tag is another void tag that breaks a line, while p or paragraph has content, after which the line breaks as well.
